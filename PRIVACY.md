# Light Reader Privacy Policy

Light Reader is a local Chrome extension for making dark reading pages easier to read.

## Data Stored

Light Reader stores settings in Chrome sync storage:

- global enabled state
- selected background, text, and link colors
- default background color
- per-site mode rules

These settings are managed by Chrome and may sync through your Chrome account if browser sync is enabled.

## Data Not Collected

Light Reader does not collect analytics, does not include tracking, does not run a backend service, and does not send page content, browsing history, or settings to any server owned by this project.

## Page Access

The extension reads the current page structure and styles locally in your browser so it can decide whether a dark reading surface should be lightened. This analysis stays on your device.

## Contact

For issues or questions, use the diagnostics in the Light Reader popup's Support panel when filing a bug report.
